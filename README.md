# TappAV Web Development Assignment

## Overview
This project includes a Home Page and a Product Page designed for TappAV, focusing on clean UI, structured layout, and responsiveness.

## Pages Included
- Home Page (index.html)
- Product Page featuring Epson EH-TW7100 projector (product.html)

## Tools & Technologies Used
- HTML5
- CSS3 (Flexbox for layout & responsiveness)
- JavaScript (basic interactivity)
- GitHub Pages for deployment

## Design Approach
I focused on a minimal and premium layout inspired by audio-visual retail websites. The design emphasizes clarity, product visibility, and ease of navigation for users.

## Responsiveness
The layout adapts for mobile and tablet devices using media queries and flexible layouts.

## Challenges Faced
Ensuring image paths worked correctly on deployment and maintaining consistent spacing across devices.
